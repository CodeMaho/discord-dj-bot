# Setup Rapido

Guia de 5 minutos para tener el bot funcionando.

## Paso 1: Instalar

```bash
git clone https://github.com/CodeMaho/discord-dj-bot.git
cd discord-dj-bot
```

Doble click en **INSTALL.bat**

## Paso 2: VB-Cable

El instalador te guiara, pero basicamente:
1. https://vb-audio.com/Cable/
2. Descargar e instalar
3. Reiniciar PC

## Paso 3: Configurar Discord

Cuenta DJ → Configuracion → Voz y Video:
- Entrada: **CABLE Output**
- Sensibilidad: **minimo**
- Cancelacion de eco: **OFF**

## Paso 4: Iniciar

Doble click en **START-DJ.bat**

## Paso 5: Usar

1. Abre http://localhost:3000 (o la URL del tunel para remoto)
2. Selecciona **CABLE Input**
3. Pega URL de YouTube
4. Click **Reproducir**
5. Une la cuenta DJ a la llamada

## Listo

La musica sonara en Discord.
